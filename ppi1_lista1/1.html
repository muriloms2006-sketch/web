A difrença entre a Internet e a Web

A Internet é uma rede global de cabos, roteadores satélites e outros componentes físicos que conectam dispositivos em todo o mundo. Utiliza protocolos como o TCP/IP para enviar e receber dados entre computadores. A Web também conhecida com World Wide Web é um sistema de informações interligadas (páginas web, imagens, vídeos) que opera sobre essa infraestrutura. Utiliza o protocolo HTTP (ou HTTPS) para acessar e exibir o conteúdo através de um navegador.

A História da Internet e Web

A Internet surgiu em 1969 com o desenvolvimento da ARPANET, uma rede experimental dos Estados Unidos para fins acadêmicos e militares durante a Guerra Fria. A primeira conexão foi estabelecida entre a Universidade da Califórnia em Los Angeles (UCLA) e o Instituto de Pesquisa de Stanford (SRI).

No início dos anos 80, a ARPANET mudou o protocolo NCP para o novo TCP/IP. Em 1985, a Internet já estava consolidada como a principal rede de comunicação com alcance global.

No final da década de 80 a internet intercone milhares de centro de pesquisa em todo o mundo, institutos, universidades e grandes empresas. A rede chegou ao Brasil em 1988, sendo liberada para uso privado e comercial em 1994.

Em 1989, Tim Berners-Lee, um físico britânico, inventou a World Wide Web enquanto trabalhava no CERN. Ele propôs um sistema de gerenciamento de informações que utilizava hipertexto para facilitar o compartilhamento e a atualização de informações entre pesquisadores.

Principais Marcos

  • 1969 - Criação da ARPANET
  • 1985 - Internet se torna global
  • 1989 - Tim Berners-Lee cria a Web
  • 1990 - Primeiro servidor e navegador da web
  • 1991 - Primeira página HTML publicada

Termos Importantes

HTTP
Protocolo que permite a comunicação entre cliente e servidor na Web.
HTML
Linguagem usada para estruturar as páginas da Web.
CSS
Define o estilo e o layout das páginas HTML.
