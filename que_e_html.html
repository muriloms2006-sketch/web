O que é HTML ?
HTML é uma linguagem que ser para estruturar e modificar as páginas da web

Exemplos do uso

  1. as tags <html> são marcadores que possibilitam declarar que elemento gráfico do browser deve ser utilizado para o texto
  2. as tags <body> marca o inicio do corpo do documento
  3. No cabeçalho <head> são trazidas informações que serão usados para o navegador como: <title> que definie o titulo do documento, <link> que define documentos a serem incorporados como outro html ou css, e <meta> que possibiita o documento a setar opções de interprtação do documenyo html.
