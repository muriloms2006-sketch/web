Formularios

O HTML possui a tag from que possibilita definir formularios para submissão de dados informados pelo usuário por meio de campos input, dentro do formúlario. A tag <form> possui além dos atributos globais e também atributo= "url", utilizando para definir o destino dos dados submetidos Exemplos: 

    <form action="url">
    <:input type="text" name="variaveila"
    ...
    </form>

Table

A tag table é utilizada para definiçãi de tabla no HTML, internamente em uma tabale podemos criar linhas "rows" utilizando a tag <tr> e celulas dentro das linhas utilizando a tag <:td> exemplo:

    <table>
        <:tr>
                <td> Celula 1x1 </td> Celula 1x2 </td>
        <tr>
        <tr>
                <td> Celula 2x1 </td> Celula 2x2 </td>
        <tr>
    </table>
O código anteior define a tabela abaixo
Celula 1x1 Celula 1x2
Celula 2x1 Celula 2x2
