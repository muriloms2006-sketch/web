Ativiadade 1 - Regra CSS

O que é uma regra CSS?

Uma regra CSSM é um conjunto de instruções que define como um ou mais elementos HTML devem ser exibidos. Ela é composta por um seletor, que indica quais elementos serão a fetados, e um bloco de declarações, que contém pares de propriedades e valores para estilizar esses elementos.

Exemplo:

p{ color: blue; font-size: 18px; }

Nesta regra:

  • Seletor: p selecionou todos os parágrafos
  • Propriedade: color mudou a cor do texto
  • Valor: blue aplicou acor azul
