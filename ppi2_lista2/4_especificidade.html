Atividade 4 - Especifidade CSS

Exemplo de conflitos entre seletores

Este parágrafo tem apenas o estilo de seleto p.

Este parágrafo tem classe destaque, que é mais forte que o seletor p.

Este parágrafo tem um estilo inline. Mesmo tendo classe e ID, o inline vence em todos os outros.

Explicação:

  • Seletor de elemento (p): 1 ponto.
  • Classe (.destaque): 10 pontos.
  • ID (#importante): 100 pontos.
  • Inline (style= "..."): 1000 pontos, vence todos.

Portanto, a cor final é escolhida pela regra mais específica. Se duas regras tiverem a mesma especificidade, o navegador aplica a que foi declarada por último no código CSS.
