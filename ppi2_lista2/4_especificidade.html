Explicação das cores e especificidade:

Abacaxi: Vermelho (li { color: red; }) — apenas o seletor de elemento foi aplicado.

Melão: Cinza (.fruta { color: gray; }) — classe tem mais peso que seletor de tag.

Pera: Azul (#lip2 { color: blue; }) — ID tem prioridade maior que classe.

Uva: Laranja (#lip3.fruta { color: orange; }) — combinação de ID + classe tem ainda mais peso.

Maçã: Cinza (.fruta { color: gray; }) — aplica apenas a classe.

Carne: Verde (.churras { color: green; }) — classe aplicada normalmente.

Linguiça: Preto (li + li.churras { color: black; }) — o seletor li + li seleciona o irmão adjacente da tag anterior.

Pão de alho: Magenta (.churras[draggable="true"] { color: magenta; }) — seletor de atributo tem maior peso que o seletor de classe isolado.
