Atividade 3 - Formas de incluir

1. CSS inline

Parágrafo estilizado usando CSS inline. As regras de estilo estão dentro do atributo "style" da própria tag.

2. CSS interno

Parágrafo usa CSS interno, declarado dentro da tag <style> no cabeçalho do documento.

3. CSS externo

Parágrafo utiliza CSS externo, aplicando a partir de um arquivo separado chamado style.css
